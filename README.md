**Lab 2: Introduction to ROS2 and Simulation Environment**

**Learning Goals**
- Understand fundamental ROS2 concepts (nodes, topics, services, packages)
- Create a basic ROS2 Python package
- Launch Gazebo from ROS2 using launch files
- Use ros_gz_bridge to connect Gazebo and ROS2 topics
- Write a ROS2 publisher node to control robot motion
- Write a ROS2 subscriber node to process LiDAR data
- Visualize sensor data in RViz2
